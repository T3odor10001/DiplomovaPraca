### Class: WeatherApp
Weather application with features to display current weather, forecast, and logging data.

#### Key Methods:

*   `create_location_section`: Creates the location input section.
*   `create_weather_display`: Creates the weather information display.
*   `create_forecast_section`: Creates the 5-day forecast display.
*   `create_datetime_display`: Creates the date and time display.
*   `create_footer`: Creates the footer with copyright information.
*   `get_weather`: Retrieves current weather data for a given location.
*   `update_weather_display`: Updates the main weather information display.
*   `update_forecast_display`: Updates the forecast display.
*   `update_temperature_display`: Refreshes the temperature display.
*   `initialize_excel_log`: Initializes the Excel logging system.

### Methods:

#### create_location_section
Parameters: None

Description: Creates the location input section with a city entry, "Get Weather" button, and temperature unit toggle.

#### create_weather_display
Parameters: None

Description: Creates the weather information display with a weather icon, temperature and condition labels, and detailed information frames.

#### create_forecast_section
Parameters: None

Description: Creates the 5-day forecast display with day labels, weather icons, temperatures, conditions, and sunrise/sunset times.

#### create_datetime_display
Parameters: None

Description: Creates the date and time display with a current date and time label.

#### create_footer
Parameters: None

Description: Creates the footer with copyright information.

#### get_weather
Parameters:
    -   `city`: The city name to retrieve weather data for.
    
Description: Retrieves current weather data for a given location using the OpenWeatherMap API. Updates the main weather information display and logs the data to Excel.

#### update_weather_display
Parameters:
    -   `data`: A dictionary containing current weather data.

Description: Updates the main weather information display with temperature, condition, humidity, wind speed, pressure, visibility, sunrise time, and sunset time.

#### update_forecast_display
Parameters:
    -   `data`: A dictionary containing forecast data for the next 5 days (every 24 hours).

Description: Updates the forecast display with day labels, temperatures, conditions, and weather icons.

#### update_temperature_display
Parameters: None

Description: Refreshes the temperature display with a new unit based on user preference.

#### initialize_excel_log
Parameters: None

Description: Initializes the Excel logging system by creating a data folder if it doesn't exist and setting up an Excel file for storing data.
